# Switch

Displays a switch or a component that looks like a switch.

<ComponentsTabs component={Switch.Default()} />

## Installation

Not required.

## Usage

<Tree.Container>
  <Tree.Root defaultExpanded>
    <Tree.Item name="Canvas" type={Box}>
      {Switch.Default({ defaultExpanded: false }).content}
      {Switch.WithTextLeft({ defaultExpanded: false }).content}
      {Switch.WithTextRight({ defaultExpanded: false }).content}
    </Tree.Item>
  </Tree.Root>
</Tree.Container>
