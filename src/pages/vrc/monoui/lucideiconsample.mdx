## Getting Started

<Steps>

### Create Lucide Icon Sample

1. Right-click in the Hierarchy window
2. Navigate to `/Mono UI/Sample/Lucide Icon Sample`
3. Click to create the Lucide Icon Sample

### Place in Scene

1. The Lucide Icon Sample will be created at the selected position in your Hierarchy
2. Adjust position and scale as needed

### Customize

1. Select the created Lucide Icon Sample
2. Modify component settings in the Inspector window
3. Add or remove components as needed
