## Getting Started

<Steps>

### Create UI Sample

1. Right-click in the Hierarchy window
2. Navigate to `/Mono UI/Sample/UI Sample`
3. Click to create the UI Sample

### Place in Scene

1. The UI Sample will be created at the selected position in your Hierarchy
2. Adjust position and scale as needed

### Customize

1. Select the created UI Sample
2. Modify component settings in the Inspector window
3. Add or remove components as needed

</Steps>

<Callout type="info">
  These samples are provided for learning purposes. When using in production, we
  recommend customizing them to meet your project requirements.
</Callout>
