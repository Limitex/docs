## Getting Started

<Steps>

### Import Sample Prefab

1. Open the `Project` window in your Unity project
2. Navigate to `/Packages/Mono UI/Runtime/Assets/Prefab/Sample` folder
3. Select the `UI Sample.prefab` prefab

### Place in Scene

1. Drag and drop the `UI Sample.prefab` prefab into your Hierarchy window
2. Adjust position and scale as needed

### Customize

1. Select the prefab instance
2. Modify component settings in the Inspector window
3. Add or remove components as needed

</Steps>

<Callout type="info">
  These samples are provided for learning purposes. When using in production, we recommend customizing them to meet your project requirements.
</Callout>
