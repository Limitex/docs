## Getting Started

<Steps>

### Create Platform Statistics

1. Right-click in the Hierarchy window
2. Navigate to `/Mono UI/Sample/Platform Statistics`
3. Click to create the Platform Statistics component

### Place in Scene

1. The Platform Statistics will be created at the selected position in your Hierarchy
2. Adjust position and scale as needed

### Customize

1. Select the created Platform Statistics
2. Modify component settings in the Inspector window
3. Add or remove components as needed

</Steps>
