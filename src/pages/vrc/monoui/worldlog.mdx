## Getting Started

<Steps>

### Create World Log

1. Right-click in the Hierarchy window
2. Navigate to `/Mono UI/Sample/World Log`
3. Click to create the World Log

### Place in Scene

1. The World Log will be created at the selected position in your Hierarchy
2. Adjust position and scale as needed

### Customize

1. Select the created World Log
2. Modify component settings in the Inspector window
3. Add or remove components as needed
