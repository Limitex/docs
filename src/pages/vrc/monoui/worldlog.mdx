## Getting Started

<Steps>

### Import Sample Prefab

1. Open the `Project` window in your Unity project
2. Navigate to `/Packages/Mono UI/Runtime/Assets/Prefab/Sample` folder
3. Select the `World Log.prefab` prefab

### Place in Scene

1. Drag and drop the `World Log.prefab` prefab into your Hierarchy window
2. Adjust position and scale as needed

### Customize

1. Select the prefab instance
2. Modify component settings in the Inspector window
3. Add or remove components as needed
