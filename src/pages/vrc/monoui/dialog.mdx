# Dialog

Displays a dialog or a component that looks like a dialog.

<ComponentsTabs component={Dialog.Default()} />

## Installation

### Place the dialog provider

Place the Dialog Provider component directly under the Canvas component in your Unity hierarchy.

<Callout>
  For proper functionality, the Dialog Provider must be placed as an immediate child of the Canvas component.
</Callout>

<Tree.Container>
  <Tree.Root defaultExpanded>
    <Tree.Item name="Canvas" type={Box}>
      {Dialog.Default({ defaultExpanded: false, variant: "highlight" }).content}
    </Tree.Item>
  </Tree.Root>
</Tree.Container>

## Usage

<Steps>

### Add the provider to the button's OnClick

Add the provider to the button's OnClick and set `MonoUI_Dialog_isTrigger` in Animator.SetTrigger (string) to call the dialog.

<Tree.Container>
  <Tree.Root defaultExpanded>
    <Tree.Item name="Canvas" type={Box}>
      {Button.Default({ defaultExpanded: false, variant: "highlight" }).content}
      {Dialog.Default({ defaultExpanded: false }).content}
    </Tree.Item>
  </Tree.Root>
</Tree.Container>


### Set up the functions for Continue and Cancel buttons

Add the functions you want to call to the OnClick events of the Continue or Cancel buttons. These can be set in `Dialog Provider/Foreground/Container/Button Container/Button Outline` or `Button` components.

<Tree.Container>
  <Tree.Root defaultExpanded>
    <Tree.Item
      name="Canvas"
      type={Box}
      highlightPaths={[
        "Canvas/Dialog Provider/Foreground/Container/Button Container/Button Outline",
        "Canvas/Dialog Provider/Foreground/Container/Button Container/Button"
      ]}
    >
      {Button.Default({ defaultExpanded: false }).content}
      {Dialog.Default({ defaultExpanded: true }).content}
    </Tree.Item>
  </Tree.Root>
</Tree.Container>

</Steps>
